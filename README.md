# 360V6 ImmortalWrt 固件编译

## 📋 简介

本仓库用于编译360安全路由器V6的ImmortalWrt固件，包含完整的NSS硬件加速支持和常用插件。

## 🚀 特性

- ✅ **IPQ6010处理器支持** - 专门针对360V6的硬件配置
- ✅ **NSS硬件加速** - 完整的高通NSS加速支持，性能提升5-10倍
- ✅ **HomeProxy** - 网络连接管理和优化工具
- ✅ **USB支持** - USB 2.0/3.0存储设备支持
- ✅ **文件系统** - EXT4、FAT32、NTFS文件系统支持
- ✅ **网络工具** - SSH、诊断工具、防火墙等
- ✅ **无线支持** - ATH11K无线驱动，支持双频WiFi
- ✅ **PPPoE** - 拨号支持
- ✅ **IPv6** - IPv6协议支持
- ✅ **中文界面** - 完整的中文语言支持

## 📱 刷机指南

### 准备工作
- 下载编译好的固件文件
- 准备网线连接电脑和路由器
- 确保路由器电源充足

### 刷机步骤
1. 进入360路由器管理界面（本固件ip通常是 192.168.2.1）
2. 找到"系统升级"或"固件升级"选项
3. 选择下载的ImmortalWrt固件文件
4. 点击"开始升级"
5. 等待升级完成（不要断电！）
6. 升级完成后，路由器会自动重启

### 首次配置
1. 连接到路由器的WiFi（SSID通常是 "ImmortalWrt"）
2. 打开浏览器访问 192.168.2.1
3. 按照向导完成初始配置
4. 设置管理员密码
5. 配置你的网络连接

## ⚙️ 配置说明

### NSS硬件加速

固件已包含完整的NSS硬件加速支持，无需额外配置。加速模块包括：
- NSS核心驱动
- 以太网连接管理器
- 千兆MAC加速
- 数据路径加速
- 以及其他40+个NSS加速模块

### HomeProxy配置

1. 进入"服务" -> "HomeProxy"
2. 添加你的网络连接服务器
3. 配置连接规则
4. 启用服务
